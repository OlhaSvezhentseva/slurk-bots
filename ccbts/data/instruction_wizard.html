Naming Legend for objects

⁃ screw: ○

⁃ washer: ◊

⁃ nut: □

⁃ bridge: ˂ (Occupies two locations)

Supported commands

⁃ pick (picking an object)

⁃ place (placing an object)

Command Syntax

/<command> (<board_name>, <location>)

Examples for pick command

/pick(s,a1)

/pick(t,b4)

Examples for place command

/place(s,a1)

/place(t,b4)

/place(t,c3,c4) bridges require both starting and ending locations
